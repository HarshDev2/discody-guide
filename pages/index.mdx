# Introduction
 
**Discody.js** is an library that makes making discord bot in javascript a lot more easier than with discord.js

## Quick Start
You can simply create an new discody.js bot by following these steps:

import { Steps } from 'nextra-theme-docs'

<Steps>
### Creating Project
```bash
npx create discody
```

### Running the bot
```bash
node main.js
```
</Steps>

If you would like to create the project from scratch, then can check the [Installation](/installation) process.
